
# Tcas 65 summary_api

## API Reference

#### Get all items

```http
  GET /
```



#### Get item

```http
  GET /id/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `int` | **Required**.  code of item to fetch |
